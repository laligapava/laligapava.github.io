# La Liga Pava — Web

Sitio estático (HTML/CSS/JS, sin build).

## Estructura
- `index.html` — página principal
- `liga-web-v3.css` — estilos
- `liga-web.js` — interacciones
- `image-slot.js` — componente de imagen
- `assets/` — imágenes usadas por la página

## Publicar con GitHub Pages
1. Sube este contenido a un repositorio (a la raíz o a `/docs`).
2. En el repo: Settings → Pages → Source → selecciona la rama y carpeta.
3. Espera unos minutos y tu web quedará publicada en la URL que indique GitHub.
